Heb je ervaring met progammeren
Ja ik heb een kleine achtergrond met een paar talen, ik ben begonnen met progammeren toen ik 14 was, toen was het alleen nog maar het aanpassen van LUA script voor de game FiveM. Later ben ik begonnen aan een discord bot met de taal JavaScript, dit was eigenlijk het eerste wat ik compleet had geschreven (natuurlijk wel met een paar tutorials). Toen ik te horen kreeg dat we weer een nieuwe stage moesten zoeken, ging ik er meteen mee bezig om een stage te vinden bij een software development bedrijf, na het zoeken van een bedrijf ben ik uitgekomen bij het bedrijf RB-Media. De eerste dag ben ik begonnen met de basis van HTML en CSS want ja dat is nodig op het moment dat je een stage gaat doen bij zo'n bedrijf, deze tutorial heb ik via w3schools dit ging redelijk gemakkelijk en heb ik binnen 7 uur afgekregen. Hierna ben ik niet gestopt en ben ik alles gaan uitzoeken hoe het werkte.
